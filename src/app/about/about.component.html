About
Important Note
This app is currently in alpha and being actively developed, meaning you will be able to sign up for a new account and use the app but may notice many changes coming down the pipes. Feel free to browse the "Explore" section of this app which should simulate a regular account for the most part. The app is expected to be completed in the coming months.
Why
The development of this app started in February 2019 when I wanted a free, easy way of keeping track of the books in my home. App development is a hobby of mine, so this project gave me something to work on when I was bored. It also serves as a proof-of-concept app where I can freely test new ideas. I hope this app can eventually be enjoyed my many bibliophiles out there.
How
The front end of this application is being developed in Angular. The backend consists of two Dockerized Java/Spring Boot API applications; one handles authentication via OAuth2 and the other handles authorized resource requests via JWT. Data is stored and retrieved in a MySQL database. Everything is hosted on Microsoft Azure.
